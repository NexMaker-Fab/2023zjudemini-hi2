Project Management based on Hexo
Oct 8, 2023

Prepare

Install Git

Go to the Git website and click Downloads:



Select the version for your system to download:



In any folder, click the right button of the mouse, the icon in the picture below indicates that the download is completed:


Set up a GitHub repository

Hexo is the framework for the project management website, but before you install it, make a repository on GitHub:



Name it <yourname>.github.io:


Binding GitHub

Start by adding the SSH key configuration on GitHub, which should come with SSH if you have Git Bash installed.
We can check if SSH is installed by opening Git Bash in any folder:



Type in this code:

ssh-keygen -t rsa

Pressing the Enter key four times will generate two files: the secret key id_rsa and the public key id_rsa.pub. Files are generated in the following directories by default:
Linux: ~/.ssh
Mac: ~/.ssh
Windows 10: C:/Users/ASUS/.ssh
Let’s copy the file and go to our GitHub home page, click on the avatar and then settings:



Create a new SSH key, paste the contents of the copied public key id_rsa.pub into the key, and click Add SSH key:


Environment Configuration

Install Hexo

Start by creating a Blog folder, open Git Bash, and install Hexo by typing npm:

npm install -g hexo-cli




Once installed, type hexo init to initialize the blog:



Then type hexo g for static deployment:



At this point, the web page has been deployed, enter the hexo s command, the browser type localhost:4000 to open the newly deployed web page for local viewing:


Deploying Hexo to GitHub

Now back in our Blog folder, open the _config.yml file and change the selected part as follows:

deploy:
type: git
repository: https://github.com/Yourname/Yourname.github.io.git #你的仓库地址
branch: main

Back in the Blog folder, open Git Bash, and enter the following command to install the Git deployment plugin:

npm install hexo-deployer-git --save

Then enter each of the following three commands:

hexo clean #清除缓存文件 db.json 和已生成的静态文件 public
hexo g #生成网站静态文件到默认设置的 public 文件夹(hexo generate 的缩写)
hexo d #自动生成网站静态文件，并部署到设定的仓库(hexo deploy 的缩写)

Then open the browser, type https://yourname.github.io can open your web page.

Installing Theme

There are many themes in Hexo, but here we introduce the Frame theme.
Open Git Bash in themes folder and enter the following command to install Frame:

git clone https://github.com/zoeingwingkei/frame.git
